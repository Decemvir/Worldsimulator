Detailed Description

Member of pattern Adapter as Adaptee. Member of pattern Singleton. Member of pattern ChainOfresponsibility as Client. This class represents the Destiny in our Universe. In our concept a system has only one instance of Destiny type. It can manage two lifeforms becoming friends evolve and devolve lifeforms. It's most important function is to create problems for life forms.
