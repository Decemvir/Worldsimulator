Detailed Description

Member of pattern Singleton.
Member of pattern Observer as ConcreteSubject.
Member of pattern Iterator. Is the list of Humans. All created Humans should be put into the List. Exceptions are clone prototypes for example. They are no "real" Humans. Extends Subject, so Observers can be attached. It will notify all Observers if an Human has been added or removed. It's the List for the Iterator HumanLifeChainIterator. Can exist only one time. This is realised using the Singleton Pattern.
