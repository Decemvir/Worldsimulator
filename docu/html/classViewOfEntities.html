Detailed Description

Member of pattern Composite as Composite. Defines an abstract graphical element that can contain other entities or views. It implements the basic functionality of Entity and adds additional features that affect every child it contains.

It gives the functionality to add to, remove from, access, clear and destroy the list of children.
