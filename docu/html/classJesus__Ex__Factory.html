Detailed Description

Member of pattern Factory as ConcreteCreator. This class can create an instance of class Jesus. Jesus_Ex_Factory is the factory for the concrete god Jesus. If this class creates a god, the static variable of Deus_Ex_Factory is set to the god created this way.
