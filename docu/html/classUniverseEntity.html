Detailed Description

Member of pattern Adapter as Adapter and Target. Member of pattern Singleton. Redefines and adds to the interface functionality inherited from the class Universe. Implements interface functionality from the class ViewOfEntities. This class defines a set of methods to render a skybox and add/remove solar systems.
