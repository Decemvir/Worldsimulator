Detailed Description

Member of pattern Composite as Leaf. Member of pattern Adapter as Adapter and Target. Redefines and adds to the interface functionality inherited from the class LifeForm. Implements interface functionality from the class Entity. This class defines a set of methods to create jobs for the life form specific job queue, start it's life and calculate the necessary three dimensional information to render it as a colored point.
