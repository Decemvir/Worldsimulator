Detailed Description

Member of pattern Facade as Subsystem classes. Member of pattern Command as Command. Defines an abstract job interface with minimal functionality so it can be used in a priority queue. It also has the basic functionality of working on it with returning true if the job is done and some methods to access or change the defined addresses and attributes.
