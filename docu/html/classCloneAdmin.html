Detailed Description

Member of pattern Singleton. Used to administrate an Singleton.
Member of pattern Prototype as Client. Sets ConcretePrototype of the HumanBreeder. Stores a Woman and a Man as "clone prototype". They can be given to the breeder. Could be extended to administrate more than one HumanBreeder. Can exist only one time. This is realised using the Singleton Pattern.
